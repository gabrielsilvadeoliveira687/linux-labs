# Linux Labs

## Objetivo
Repositório de estudo e prática em Linux, organizado por tópicos.

## Estrutura
- 01-introducao: Conceitos, distribuições e histórico
- 02-gerenciamento-pacotes: apt, yum, dnf
- 03-comandos-basicos: Navegação, arquivos, pesquisa
- 04-permissoes-usuarios: Usuários, grupos e permissões
- 05-servicos-processos: Processos, serviços e logs
- 06-redes: IP, ping, traceroute, SSH e firewall
- 07-shell-scripting: Scripts básicos, loops, if/else
- 08-notas-avancadas: Comandos avançados, cron, rsync e boas práticas

## Como usar
1. Navegue até a pasta desejada
2. Leia o README.md dentro da pasta
3. Execute os exemplos no terminal
